Family Invite Flow (New)
1
Player completes onboarding step 5
Enters parent email → POST /api/family/invite
2
Invite email sent via Resend
Token stored in family_invitations, 7-day TTL
3
Parent opens link (web or iOS universal)
pages/join.vue — reads token, shows invite details
4
Parent accepts → signs up or signs in
POST /api/family/invite/[token]/accept — links accounts
✓
Family unit established
Parent enters 5-step onboarding wizard, gains read access
