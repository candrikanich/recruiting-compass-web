- **Investigation Reports** - Root cause analyses and diagnostic findings
- **Implementation Notes** - Feature implementation details and patterns discovered

- **Testing/Validation** - Test failure guides and validation procedures
- **Setup Guides** - Historical setup documentation (Vercel migration, etc.)
